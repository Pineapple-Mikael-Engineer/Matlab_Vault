# 01 Entorno y Flujo de Trabajo

Capítulo de base operativa: cómo configurar MATLAB, escribir código, depurar y documentar.

## Objetivo del capítulo

Dominar un flujo de trabajo reproducible desde la edición del código hasta su publicación.

## Secciones

- [[1.1 Interfaz/index|1.1 Interfaz]]: componentes del entorno (Editor, Command Window, Workspace, Current Folder).
- [[1.2 Configuración de la ruta de Búsqueda/index|1.2 Ruta de búsqueda]]: `addpath`, `savepath` y `pathtool`.
- [[1.3 Scripts vs funciones/index|1.3 Scripts vs funciones]]: estructura del código y organización modular.
- [[1.4 Depuración/index|1.4 Depuración]]: breakpoints y comandos `db*`.
- [[1.5 Publicación y documentación/index|1.5 Publicación y documentación]]: comentarios, celdas y `publish`.

## Dependencias entre secciones

- 1.1 y 1.2 habilitan el entorno.
- 1.3 define cómo escribes el código.
- 1.4 valida/corrige errores.
- 1.5 comunica resultados y documentación.
