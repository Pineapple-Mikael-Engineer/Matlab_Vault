# Cadenas y caracteres

MATLAB ofrece dos formas de representar texto: arreglos de caracteres (`char`) y cadenas (`string`). Las primeras son la forma clásica; las segundas fueron introducidas en R2016b y son más convenientes para la mayoría de los casos.

## Arreglos de caracteres (`char`)

Usan comillas simples `''`. Cada carácter ocupa 2 bytes (UTF-16).

```matlab
% Creación
c1 = 'Hola';
c2 = 'Mundo';
size(c1)           % 1×4 (es un arreglo)

% Concatenación horizontal
c3 = [c1, ' ', c2];   % 'Hola Mundo'

% Concatenación vertical (matriz de caracteres, todas filas igual largo)
c4 = ['abc'; 'def'];   % 2×3 char

% Acceso a caracteres
c1(2)                % 'o'
c1(end)              % 'a'

% Arreglo de caracteres vs string
whos c1              % 1×4 char
```

## Cadenas (`string`)

Usan comillas dobles `""`. Son más fáciles de manejar y permiten almacenar texto de diferentes longitudes.

```matlab
% Creación
s1 = "Hola";
s2 = "Mundo";
size(s1)           % 1×1 (es un escalar string)

% Concatenación
s3 = s1 + " " + s2;   % "Hola Mundo"

% Arreglo de strings
s_arr = ["Hola", "Mundo"; "Foo", "Bar"];   % 2×2 string

% Conversión entre tipos
c = 'texto';
s = string(c);     % char → string
c2 = char(s);      % string → char

% Verificar tipo
ischar(c)          % true
isstring(s)        % true
```

## Comparación de `char` y `string`

| Operación | `char` | `string` |
|-----------|--------|----------|
| Delimitadores | Comillas simples `''` | Comillas dobles `""` |
| Tipo de dato | Arreglo 1×N | Escalar o arreglo |
| Concatenación | `[c1, c2]` o `strcat` | `s1 + s2` |
| Indexación | Por carácter | Como elemento |
| Longitud variable | No (matriz rectangular) | Sí |
| Uso recomendado | Datos legacy, caracteres sueltos | Texto general |

## Funciones fundamentales para strings

### `strcat` — concatenación horizontal
```matlab
% Para char (ignora espacios al final)
s = strcat('Hola', ' ', 'Mundo');   % 'HolaMundo' (sin espacio)

% Para string
s2 = strcat("Hola", " ", "Mundo");  % "Hola Mundo"

% Mejor usar + para strings
s3 = "Hola" + " " + "Mundo";
```

### `split` — dividir string
```matlab
texto = "Hola,Mundo,MATLAB";
partes = split(texto, ",");      % ["Hola"; "Mundo"; "MATLAB"]

% Con char
texto2 = 'Hola Mundo MATLAB';
partes2 = split(texto2);          % ["Hola"; "Mundo"; "MATLAB"]
```

### `join` — unir strings
```matlab
partes = ["Hola", "Mundo", "MATLAB"];
unido = join(partes, " ");       % "Hola Mundo MATLAB"
unido2 = join(partes, " - ");    % "Hola - Mundo - MATLAB"
```

### `contains` — verificar presencia de patrón
```matlab
textos = ["gato", "perro", "gaviota", "pez"];
contiene_g = contains(textos, "g");     % [true, false, true, false]
contiene_ga = contains(textos, "ga");   % [false, false, true, false]

% Con múltiples patrones
contiene = contains(textos, ["gato", "pez"]);  % [true, false, false, true]
```

### `replace` / `replaceBetween` — reemplazar texto
```matlab
texto = "El gato come pescado";
nuevo = replace(texto, "gato", "perro");      % "El perro come pescado"

% Múltiples reemplazos
texto2 = "manzana,perro,rojo";
nuevo2 = replace(texto2, ["perro", "rojo"], ["gato", "azul"]);  
% "manzana,gato,azul"

% Reemplazar por posición
texto3 = "abcdef";
nuevo3 = replaceBetween(texto3, 2, 4, "XYZ"); % "aXYZef"
```

## Funciones adicionales importantes

| Función | Descripción | Ejemplo |
|---------|-------------|---------|
| `strlength` | Longitud del string | `strlength("Hola")` → 4 |
| `lower` / `upper` | Minúsculas/mayúsculas | `lower("HOLA")` → "hola" |
| `strcmp` | Comparación exacta | `strcmp("Hola","Hola")` → true |
| `strcmpi` | Comparación insensible a mayúsculas | `strcmpi("Hola","hola")` → true |
| `startsWith` / `endsWith` | Verifica inicio/final | `startsWith("gato","ga")` → true |
| `erase` | Elimina substrings | `erase("Hola Mundo","Mu")` → "Hola ndo" |
| `extractBetween` | Extrae entre delimitadores | `extractBetween("abc123def","bc","ef")` → "123" |
| `strip` | Elimina espacios | `strip("  Hola  ")` → "Hola" |

## Conversión entre texto y números

```matlab
% Número → texto
num2str(3.1416)          % "3.1416"
num2str(pi, '%.2f')      % "3.14"
string(123)              % "123"

% Texto → número
str2double("3.14")       % 3.14
str2double("3.14e-2")    % 0.0314
str2double("no numero")  % NaN

% Formateo avanzado
sprintf('Valor = %.3f', pi)    % "Valor = 3.142"
sscanf('3.14 2.71', '%f %f')   % [3.14; 2.71]
```

## Patrones (pattern) para búsqueda avanzada

```matlab
% Desde R2020b, patrones más legibles
texto = "El archivo imagen_001.jpg se guardó";

% Patrones predefinidos
digitsPattern    % coincide con dígitos
lettersPattern   % coincide con letras
whitespacePattern % espacios

% Ejemplo práctico
numeros = extract(texto, digitsPattern);      % "001"
extension = extractAfter(texto, "imagen_");   % "001.jpg"

% Patrones personalizados
pat = lettersPattern + digitsPattern;         % "imagen001"
resultado = extract(texto, pat);              % "imagen001"
```
