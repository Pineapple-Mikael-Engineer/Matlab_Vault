# Tipos numéricos y lógicos

MATLAB maneja diferentes tipos numéricos y el tipo lógico para operaciones booleanas. Por defecto, todos los números son `double`.

## Tipos de punto flotante

| Tipo | Bytes | Rango | Precisión |
|------|-------|-------|-----------|
| `double` | 8 | ±1.7e±308 | 15-17 dígitos |
| `single` | 4 | ±3.4e±38 | 6-9 dígitos |

```matlab
% Por defecto es double
x = 3.14;        % double
class(x)         % 'double'

% Conversión explícita
y = single(3.14); % single
z = double(y);    % double

% Verificar tipo
isfloat(x)       % true (double y single son float)
isdouble(x)      % true
issingle(y)      % true
```

## Tipos enteros

| Tipo | Rango | Bytes |
|------|-------|-------|
| `int8` | -128 a 127 | 1 |
| `uint8` | 0 a 255 | 1 |
| `int16` | -32768 a 32767 | 2 |
| `uint16` | 0 a 65535 | 2 |
| `int32` | -2.1e9 a 2.1e9 | 4 |
| `uint32` | 0 a 4.3e9 | 4 |
| `int64` | -9.2e18 a 9.2e18 | 8 |
| `uint64` | 0 a 1.8e19 | 8 |

```matlab
% Creación
a = int8(127);     % Máximo valor int8
b = uint8(255);    % Máximo valor uint8

% Desbordamiento (saturación)
c = int8(128);     % 127 (satura al máximo)
d = uint8(-1);     % 0 (satura al mínimo)

% Conversión
e = int16(30000);
f = uint16(e);

% Verificar tipo
isinteger(a)       % true
```

## Valores especiales

### `NaN` (Not a Number)
Representa resultados matemáticos indefinidos.

```matlab
NaN_resultados = [0/0, inf/inf, NaN + 5];
isnan(NaN_resultados)  % [true, true, true]

% Propagación: cualquier operación con NaN da NaN
NaN + 10        % NaN
NaN * 0         % NaN

% Funciones útiles
datos = [1 2 NaN 4 5];
media = mean(datos, 'omitnan');      % Ignora NaN
datos_limpios = rmmissing(datos);    % [1 2 4 5]
```

### `Inf` e `-Inf` (Infinito)
Representa desbordamiento numérico o división por cero.

```matlab
Inf_resultados = [1/0, exp(1000), realmax*2];
isinf(Inf_resultados)  % [true, true, true]

% Propiedades
Inf + 5         % Inf
Inf - Inf       % NaN
1/Inf           % 0
log(0)          % -Inf

% Verificar
isinf(x)        % true si es Inf o -Inf
isfinite(x)     % true si no es Inf ni NaN
```

### `eps` (Épsilon de máquina)
Distancia entre 1.0 y el siguiente número representable.

```matlab
eps              % 2.2204e-16 (para double)
eps('single')    % 1.1921e-07

% Uso práctico: comparación de números flotantes
a = 0.1 + 0.1 + 0.1;
b = 0.3;
abs(a - b) < eps  % true (comparación segura)
```

## Tipo lógico (`logical`)

Representa valores booleanos `true` (1) y `false` (0).

```matlab
% Creación directa
v = [true, false, true];   % 1×3 logical
f = false(2, 3);           % Matriz 2×3 de false

% Conversión desde numérico
x = [1 0 2 0 5];
logical(x)                 % [true, false, true, false, true]

% Operadores relacionales devuelven logical
A = [1 2 3; 4 5 6];
mascara = A > 3;           % [false, false, false; true, true, true]
class(mascara)             % 'logical'

% Operadores lógicos
a = true;
b = false;
a & b   % false (AND)
a | b   % true  (OR)
~a      % false (NOT)
xor(a, b) % true  (XOR)

% Para cortocircuito usar && y ||
if (x > 0 && y < 10)   % Evalúa segundo solo si primero es true
```

## Conversión entre tipos

```matlab
% Conversión explícita
double(5)      % 5
single(3.14)   % 3.1400
int32(3.7)     % 4 (redondea)
uint8(-5)      % 0 (satura)
logical(0)     % false
logical(5)     % true

% Funciones de verificación
isnumeric(x)   % true para double, single, enteros
isfloat(x)     % true para double, single
isinteger(x)   % true para enteros
islogical(x)   % true para logical
isa(x, 'double') % verifica tipo específico
```

## Redondeo y funciones numéricas

```matlab
% Redondeo
round(3.6)     % 4
floor(3.6)     % 3
ceil(3.2)      % 4
fix(3.9)       % 3 (hacia cero)

% Límites numéricos
realmax        % 1.7977e+308 (máximo double)
realmin        % 2.2251e-308 (mínimo positivo double)
intmax('int32') % 2147483647
intmin('int32') % -2147483648
```
