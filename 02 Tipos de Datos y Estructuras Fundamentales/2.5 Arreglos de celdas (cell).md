# Arreglos de celdas (cell)

Los arreglos de celdas son contenedores que pueden almacenar cualquier tipo de dato en cada una de sus posiciones: números, strings, matrices, otros cell arrays, structs, incluso funciones. Son la solución en MATLAB para trabajar con datos heterogéneos.

## Creación

### Con llaves `{}`

```matlab
% Creación manual
C = {1, 'texto', [1 2; 3 4], true}     % 1×4 cell

% Creación por filas
C2 = {1 2 3; 'a' 'b' 'c'; rand(2) magic(3) pi}  % 3×3 cell

% Cell vacío
vacio = {}                             % 0×0 cell
fila_vacia = cell(1, 5)                % 1×5 cell (todos [])
```

### Con función `cell()`

```matlab
% Preasignación (recomendado para grandes arreglos)
C = cell(3, 4);        % 3×4 cell, todos vacíos
C = cell(5, 1);        % 5×1 cell

% Inicialización
C{1,1} = [1 2 3];
C{2,1} = "texto";
```

## Indexación: `()` vs `{}`

La diferencia es fundamental y causa común de errores.

### Paréntesis `()` — subarreglo de celdas

Devuelve **otro cell array** (subconjunto).

```matlab
C = {1, 'texto', [1 2; 3 4], true};

sub = C(2:3)          % 1×2 cell: {'texto', [1 2; 3 4]}
class(sub)            % 'cell'
```

### Llaves `{}` — contenido de las celdas

Devuelve el **contenido real** almacenado en las celdas.

```matlab
C = {1, 'texto', [1 2; 3 4], true};

contenido = C{2}      % 'texto' (string o char)
class(contenido)      % 'char' o 'string'

% Acceso a múltiples celdas
[a, b] = C{1:2}       % a = 1, b = 'texto'
```

### Tabla comparativa

| Operación | Resultado | Tipo |
|-----------|-----------|------|
| `C(1,2)` | Subconjunto de celdas | `cell` |
| `C{1,2}` | Contenido de una celda | Variable |
| `C{1,2}(3)` | Contenido, luego indexación | Variable |

```matlab
C = {[10 20 30], 'Hola'};

C(1)        % {[10 20 30]}  → cell 1×1
C{1}        % [10 20 30]     → double 1×3
C{1}(2)     % 20             → double
```

## Asignación

```matlab
C = cell(2, 3);

% Asignar con llaves (al contenido)
C{1,1} = [1 2 3];
C{2,3} = 'texto';
C{1,2} = @sin;           % función anónima

% Asignar con paréntesis (a subarreglo)
C(2,1:2) = {magic(3), true};   % asignar múltiples celdas
```

## Almacenamiento de datos heterogéneos

```matlab
% Una misma estructura de cell puede mezclar cualquier tipo
datos = {...
    42,                         % double
    "Juan Pérez",               % string
    [2024, 2025, 2026],         % vector
    struct('edad', 30, 'peso', 75),  % struct
    @(x) x^2,                   % function handle
    {'subcell', 'anidado'}      % otro cell
};

% Acceder a elementos anidados
datos{6}{1}          % 'subcell'
```

## Funciones útiles para cell arrays

| Función | Descripción | Ejemplo |
|---------|-------------|---------|
| `cellfun` | Aplicar función a cada celda | `cellfun(@isnumeric, C)` |
| `celldisp` | Mostrar contenido | `celldisp(C)` |
| `cellplot` | Visualización gráfica | `cellplot(C)` |
| `iscell` | Verificar si es cell | `iscell(C)` |
| `iscellstr` | Verificar si contiene solo strings | `iscellstr(C)` |
| `num2cell` | Convertir matriz a cell | `num2cell([1 2; 3 4])` |
| `mat2cell` | Dividir matriz en cell | `mat2cell(A, [1,1], 3)` |

## `cellfun` — operaciones sobre todas las celdas

```matlab
C = {1, 2, 3, 4, 5};

% Sumar 10 a cada elemento
C2 = cellfun(@(x) x + 10, C, 'UniformOutput', false);

% Verificar tipo
es_num = cellfun(@isnumeric, C);   % [true, true, true, true, true]

% Obtener longitudes
longitudes = cellfun(@length, C);   % [1, 1, 1, 1, 1]

% Convertir a arreglo (si todos del mismo tipo)
numeros = cellfun(@double, C);      % [1, 2, 3, 4, 5]
```

## Conversión entre cell y otros tipos

```matlab
% Matriz → cell
M = [1 2; 3 4];
C = num2cell(M)           % {[1], [2]; [3], [4]}
C = num2cell(M, 1)        % por columnas: {[1;3], [2;4]}

% Cell → matriz (si todos numéricos y mismo tamaño)
C = {1, 2; 3, 4};
M = cell2mat(C)           % [1 2; 3 4]

% Cell de strings → string array
C = {'a', 'b', 'c'};
S = string(C)             % ["a", "b", "c"]

% String array → cell
S = ["a", "b", "c"];
C = cellstr(S)            % {'a', 'b', 'c'}
```

## Casos de uso comunes

### Almacenar texto de diferentes longitudes

```matlab
nombres = {'Ana', 'Juan Carlos', 'Luz', 'Fernando'};
% Cada nombre tiene diferente longitud, no se puede en matriz char
```

### Datos importados con columnas mezcladas

```matlab
% Importar CSV con números y texto
raw_data = {'Juan', 25, 1.75; 'Ana', 30, 1.65; 'Luis', 28, 1.80};
% Luego convertir columnas según tipo
```

### Agrupar resultados de diferentes tamaños

```matlab
% Cada grupo tiene diferente número de elementos
grupos = {};
for i = 1:10
    grupos{i} = rand(1, randi(5));  % cada grupo tamaño diferente
end
```

### Pasar argumentos variables a funciones

```matlab
% Útil para funciones con número variable de argumentos
args = {'LineWidth', 2, 'Color', 'red', 'Marker', 'o'};
plot(x, y, args{:});   % args{:} expande a argumentos individuales
```

## Lista de celdas (`cell`) vs string array

| Característica | `cell` | `string` |
|----------------|--------|----------|
| Sintaxis | `{}` | `""` |
| Texto individual | `{'texto'}` | `"texto"` |
| Indexación | `C{1}` | `S(1)` |
| Operaciones de texto | Funciones antiguas | Métodos directos |
| Uso recomendado | Datos heterogéneos | Texto puro |

Para almacenar texto de forma más moderna, considerar usar [[Cadenas y caracteres|string arrays]]. Para datos tabulares heterogéneos, [[Tablas (table)|tablas]] son más adecuadas.
