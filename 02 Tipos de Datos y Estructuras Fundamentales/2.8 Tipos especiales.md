# Tipos especiales

Además de los tipos numéricos, cadenas, fechas, cell arrays, estructuras y tablas, MATLAB dispone de tipos más específicos para casos particulares: `categorical` para datos categóricos, `function_handle` para referenciar funciones, y `containers.Map` para diccionarios clave-valor.

## `categorical` — datos categóricos

Representa datos que toman valores de un conjunto finito de categorías (ej: colores, géneros, niveles educativos). Es más eficiente en memoria que strings y permite definir órdenes entre categorías.

### Creación

```matlab
% Desde string o cell
colores = categorical(["rojo", "azul", "rojo", "verde", "azul"]);

% Con niveles específicos (controla el orden)
niveles = categorical(["bajo", "alto", "medio", "alto"], ...
    {'bajo', 'medio', 'alto'});

% Con orden ordinal
temp = categorical(["frio", "caliente", "templado", "frio"], ...
    {'frio', 'templado', 'caliente'}, 'Ordinal', true);
```

### Operaciones comunes

```matlab
% Ver categorías
categories(colores)        % {'azul', 'rojo', 'verde'}

% Conteo por categoría
countcats(colores)         % [2, 2, 1]

% Tabla de frecuencias
tabulate(colores)

% Comparación (con ordinal)
temp >= 'templado'         % [false, true, true, false]

% Reordenar categorías
colores = reordercats(colores, {'verde', 'azul', 'rojo'});

% Combinar categorías
colores = mergecats(colores, {"azul", "verde"}, "azul_verde");

% Eliminar categorías no usadas
colores = removecats(colores);
```

### Ventajas sobre strings

```matlab
% Más eficiente en memoria y velocidad
datos = randi(3, 1000000, 1);
categorias = categorical(datos, [1 2 3], {'A', 'B', 'C'});

% Para análisis por grupo
T = table(categorias, rand(1000000, 1), 'VariableNames', {'Grupo', 'Valor'});
resumen = groupsummary(T, 'Grupo', 'mean', 'Valor');
```

## `function_handle` — manejadores de funciones

Permiten almacenar una referencia a una función, pasarla como argumento, o crearla de forma anónima. Son fundamentales para programación funcional.

### Creación

```matlab
% Función existente (con @)
f1 = @sin;
f2 = @mean;
f3 = @(x) x^2;           % Función anónima

% Función anónima con múltiples argumentos
area = @(radio, altura) pi * radio^2 * altura;

% Función anónima con múltiples líneas (usando helper)
calculo = @(x) deal(x^2, sqrt(x), x^3);
```

### Uso básico

```matlab
% Llamar al manejador
f1(pi/2)                 % 1
f3(5)                    % 25
area(3, 10)              % 282.743

% Pasar como argumento a otra función
cellfun(@sin, {0, pi/2, pi})           % {0, 1, 0}
arrayfun(@(x) x^2, 1:5)                % [1, 4, 9, 16, 25]

% Guardar en estructuras o tablas
ops.multiplicar = @(a,b) a * b;
ops.dividir = @(a,b) a / b;
ops.multiplicar(4, 5);                 % 20
```

### Funciones comunes que usan function_handle

| Función | Descripción |
|---------|-------------|
| `cellfun` | Aplica función a cada celda |
| `arrayfun` | Aplica función a cada elemento de arreglo |
| `structfun` | Aplica función a cada campo de estructura |
| `varfun` | Aplica función a cada columna de tabla |
| `fzero` | Encuentra raíz de función |
| `integral` | Integra función numéricamente |
| `optimoptions` | Configura optimizaciones |

```matlab
% Ejemplos prácticos
f = @(x) x^3 - 2*x - 5;
raiz = fzero(f, 2);                    % 2.0946

integral(@(x) exp(-x^2), -Inf, Inf)    % 1.7725 (sqrt(pi))

% Múltiples salidas con varfun
T = table([1;2;3], [4;5;6], 'VariableNames', {'A', 'B'});
resultados = varfun(@(x) [mean(x), std(x)], T);
```

### Funciones anónimas con parámetros fijos (currying)

```matlab
% Fijar parámetros
potencia = @(n) @(x) x^n;
cuadrado = potencia(2);
cubo = potencia(3);

cuadrado(4)              % 16
cubo(4)                  % 64

% Con parámetros de configuración
config = @(tol, maxiter) @(x) buscar(x, tol, maxiter);
buscar_fino = config(1e-6, 1000);
```

### Verificar y obtener información

```matlab
% Verificar si es function_handle
isa(f1, 'function_handle')   % true
isfunction(f1)               % true (requiere toolbox)

% Obtener nombre (si no es anónima)
func2str(@sin)               % 'sin'

% Mostrar contenido (para anónimas)
functions(@(x) x^2)          % Muestra la expresión
```

## `containers.Map` — diccionarios clave-valor

Almacena pares clave-valor, similar a un diccionario en otros lenguajes. Las claves pueden ser números o strings; los valores pueden ser cualquier tipo de dato.

### Creación

```matlab
% Con claves string
dic = containers.Map({'Ana', 'Juan', 'Luis'}, {25, 30, 28});

% Con claves numéricas
dic2 = containers.Map([101, 102, 103], {'Ana', 'Juan', 'Luis'});

% Creación vacía
dic3 = containers.Map('KeyType', 'char', 'ValueType', 'any');
```

### Operaciones básicas

```matlab
% Acceso
edad_ana = dic('Ana');           % 25
edad_juan = dic('Juan');          % 30

% Asignar o modificar
dic('Maria') = 35;                % Añade nueva clave
dic('Ana') = 26;                  % Modifica valor existente

% Eliminar
remove(dic, 'Luis');

% Verificar existencia
isKey(dic, 'Pedro')               % false
isKey(dic, 'Ana')                 % true

% Tamaño
length(dic)                       % número de pares
```

### Recorrer diccionarios

```matlab
% Obtener todas las claves y valores
claves = keys(dic);               % {'Ana', 'Juan', 'Maria'}
valores = values(dic);            % {26, 30, 35}

% Iterar
nombres = keys(dic);
for i = 1:length(dic)
    nombre = nombres{i};
    edad = dic(nombre);
    fprintf('%s tiene %d años\n', nombre, edad);
end

% Usando cellfun
edades = cellfun(@(k) dic(k), keys(dic));
```

### Almacenar tipos heterogéneos

```matlab
% Valores pueden ser cualquier cosa
registro = containers.Map('KeyType', 'char', 'ValueType', 'any');

registro('nombre') = "Ana";
registro('edad') = 25;
registro('fecha') = datetime(2026, 4, 18);
registro('datos') = [1, 2, 3, 4];
registro('funcion') = @(x) x^2;

% Acceder
registro('nombre')                % "Ana"
registro('funcion')(5)            % 25
```

### Uso práctico

```matlab
% Contador de frecuencias
textos = ["rojo", "azul", "rojo", "verde", "azul", "rojo"];
frec = containers.Map('KeyType', 'char', 'ValueType', 'double');

for t = textos
    if isKey(frec, t)
        frec(t) = frec(t) + 1;
    else
        frec(t) = 1;
    end
end
% frec('rojo') = 3, frec('azul') = 2, frec('verde') = 1

% Configuración
config = containers.Map();
config('umbral') = 0.95;
config('max_iter') = 1000;
config('metodo') = 'gradiente';
config('verbose') = true;
```

## Comparación de tipos

| Tipo | Cuándo usar |
|------|-------------|
| `categorical` | Datos con categorías fijas (ej: género, nivel educativo) |
| `function_handle` | Pasar funciones como argumentos, integración numérica, callbacks |
| `containers.Map` | Diccionarios clave-valor, búsquedas rápidas por clave no numérica |

Para almacenar datos categóricos en tablas, ver [[Tablas]]. Para usar function_handle con estructuras, ver [[Estructuras (struct)]]. Para diccionarios dentro de cell arrays, ver [[Arreglos de celdas (cell)]].
