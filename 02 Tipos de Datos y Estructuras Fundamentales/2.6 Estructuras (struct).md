# Estructuras (struct)

Las estructuras son arreglos de MATLAB que contienen datos en **campos** con nombre. Cada campo puede almacenar cualquier tipo de dato, y un arreglo de estructuras permite agrupar múltiples registros con los mismos campos.

## Creación de estructuras

### Con notación de punto

```matlab
% Creación campo por campo
persona.nombre = "Ana";
persona.edad = 30;
persona.peso = 65.5;
persona.telefonos = [123456, 789012];
persona.domicilio.ciudad = "Madrid";   % Estructura anidada
persona.domicilio.calle = "Mayor";

% Resultado: estructura escalar 1×1
```

### Con función `struct()`

```matlab
% struct(campo1, valor1, campo2, valor2, ...)
persona = struct('nombre', "Ana", 'edad', 30, 'peso', 65.5);

% Múltiples valores → arreglo de estructuras
estudiantes = struct('nombre', {"Ana", "Juan", "Luis"}, 'edad', {25, 22, 28});
% 1×3 struct con campos 'nombre' y 'edad'
```

### Estructura vacía

```matlab
vacia = struct();        % 0×0 struct
campo_vacio = struct('a', {}, 'b', {});  % 0×0 con campos definidos
```

## Acceso a campos

### Notación de punto

```matlab
persona.nombre    % "Ana"
persona.edad      % 30
persona.domicilio.ciudad  % "Madrid" (anidado)
```

### `getfield` y `setfield`

```matlab
% Alternativa para nombres de campo dinámicos
campo = 'nombre';
valor = getfield(persona, campo);     % "Ana"
persona = setfield(persona, 'edad', 31);

% Con nombres dinámicos anidados
persona = setfield(persona, 'domicilio', 'pais', 'España');
```

### `rmfield` — eliminar campos

```matlab
persona = rmfield(persona, 'telefonos');  % Elimina el campo
```

## Arreglos de estructuras

Son estructuras con múltiples elementos (como una tabla pero más flexible).

```matlab
% Creación directa
alumnos(1).nombre = "Ana";
alumnos(1).nota = 8.5;
alumnos(2).nombre = "Juan";
alumnos(2).nota = 7.0;
alumnos(3).nombre = "Luis";
alumnos(3).nota = 9.2;

% Ahora alumnos es 1×3 struct

% Acceso a elementos individuales
alumnos(2).nombre   % "Juan"
alumnos(2)          % estructura completa del elemento 2

% Acceso a todos los valores de un campo (devuelve comma-separated list)
nombres = {alumnos.nombre};    % {'Ana', 'Juan', 'Luis'}
notas = [alumnos.nota];        % [8.5, 7.0, 9.2]

% Modificar un campo para todos
[alumnos.nota] = [8.5, 7.0, 9.2];  % Asignación múltiple
```

## Campos dinámicos

Permiten acceder a campos cuyo nombre se determina en tiempo de ejecución.

```matlab
% Usando paréntesis en lugar de punto
campo_actual = 'nombre';
valor = persona.(campo_actual);      % "Ana"

campo_actual = 'edad';
valor = persona.(campo_actual);      % 30

% Útil en bucles
campos = {'nombre', 'edad', 'peso'';
for i = 1:length(campos)
    valor = persona.(campos{i});
    disp(valor);
end

% Crear campos dinámicos
nuevo_campo = 'fecha_nacimiento';
persona.(nuevo_campo) = datetime(1995, 5, 15);
```

## Funciones útiles para estructuras

| Función | Descripción | Ejemplo |
|---------|-------------|---------|
| `fieldnames` | Lista de nombres de campos | `fieldnames(persona)` |
| `isfield` | Verificar si existe campo | `isfield(persona, 'edad')` |
| `isstruct` | Verificar si es estructura | `isstruct(persona)` |
| `orderfields` | Ordenar campos alfabéticamente | `persona = orderfields(persona)` |
| `struct2cell` | Convertir campos a cell array | `celda = struct2cell(persona)` |
| `cell2struct` | Cell array a estructura | `struct = cell2struct(celda, campos, 1)` |
| `structfun` | Aplicar función a cada campo | `structfun(@class, persona, 'UniformOutput', false)` |

## `structfun` — operaciones sobre campos

```matlab
persona = struct('nombre', "Ana", 'edad', 30, 'peso', 65.5);

% Obtener tipo de cada campo
tipos = structfun(@class, persona, 'UniformOutput', false);
% {'string', 'double', 'double'}

% Sumar 5 a todos los campos numéricos
solo_numericos = structfun(@(x) x + 5, persona, 'UniformOutput', false);
```

## Estructuras anidadas

```matlab
% Creación directa
empleado.nombre = "Carlos";
empleado.direccion.calle = "Av. Principal";
empleado.direccion.ciudad = "Barcelona";
empleado.direccion.codigo = 08001;
empleado.contacto.email = "carlos@email.com";
empleado.contacto.telefono = 123456789;

% Acceso
empleado.direccion.ciudad      % "Barcelona"
empleado.contacto.email        % "carlos@email.com"

% Anidación en arreglos
departamentos(1).nombre = "Ventas";
departamentos(1).jefe.nombre = "Ana";
departamentos(1).jefe.antiguedad = 5;
departamentos(2).nombre = "TI";
departamentos(2).jefe.nombre = "Luis";
departamentos(2).jefe.antiguedad = 3;
```

## Conversión con tablas

```matlab
% Estructura → tabla
alumnos = struct('nombre', {"Ana","Juan"}, 'nota', {8.5,7.0});
T = struct2table(alumnos);   % tabla 2×2

% Tabla → estructura
alumnos2 = table2struct(T);   % recupera estructura
```

## Casos de uso comunes

### Agrupar datos relacionados sin tabla

```matlab
% Parámetros de simulación

```

## Estructuras vs tablas vs cell

| Tipo | Cuándo usar |
|------|-------------|
| `struct` | Datos heterogéneos, campos con nombre, pocos elementos |
| `table` | Datos tabulares, operaciones estadísticas, grandes volúmenes |
| `cell` | Datos heterogéneos sin nombres de campo |

Para datos tabulares con múltiples registros, [[Tablas (table)|tablas]] ofrecen más funcionalidad. Para datos heterogéneos sin nombres, [[Arreglos de celdas (cell)|cell arrays]] son más simples.
